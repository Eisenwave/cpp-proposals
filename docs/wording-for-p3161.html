This paper provides wording for [P3161R4]. This is partially motivated by the fact that wording in [P3161R4] is formatted entirely in monospace font, making it hard for the editors of the working draft to properly convert into LaTeX. [P3161R4] also contains some editorial mistakes like missing semicolons or unnecessary freestanding comments, which are fixed here.

1. Wording

All changes are relative to [N5008].
